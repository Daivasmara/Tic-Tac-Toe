# Vue.js Tic-Tac-Toe Game!

A simple Tic-Tac-Toe Game created with only Vue.js!

### How to Run this App
```
npm install

```

### Compiles and Minifies for Production
```
npm run build
```

### Lints and Fixes Files
```
npm run lint
```
