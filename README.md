# Vue.js Tic-Tac-Toe Game!

A simple Tic-Tac-Toe game created with only Vue.js!
The App is deployed here https://dist-jrjraeqeva.now.sh/ Play it Live!


### How to Run this App
```
npm install
npm run serve
```

### Compiles and Minifies for Production
```
npm run build
```

### Lints and Fixes Files
```
npm run lint
```
